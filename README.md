**PROJECT TITLE:**

TRENDS IN DEMOGRAPHICS AND INCOME

**DESCRIPTION:**

This is an interactive RSHINY dashboard that shows which hypothetical demographics (age, education, gender, etc) to target for advertising by a company that provides career services to those with lower incomes in the US. 
It explores census data to figure out current trends in different demographics that can be helpful to the marketing team.
The data originates from the census database in the excel file (adult.csv).


**Features include:**

•	Dropdown selectInput for Country 
•	a fluidRow to control how to plot the continuous variables in UI – age and hours_per_week
•	 a fluidRow to control how to plot the categorical variables in UI – education, workclass, sex.
•	 Option between histogram or boxplot to age and hours_per_week variables
•	Option of faceted bar chart and stacked bar chart for the categorical variables

**Technologies Used:**
RStudio
